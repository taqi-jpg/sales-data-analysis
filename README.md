# 🛒 Sales Data Analysis — SQL + Python

## 📌 Project Overview
Analysis of sales data using SQL queries and Python visualizations to extract business insights.

## 🛠️ Tools Used
- Python (Pandas, Matplotlib, Seaborn)
- SQL (SQLite)
- Jupyter Notebook

## 📊 Key Insights
- Electronics dominates with $3,820 in total sales
- Laptop is the best product with 25% profit margin
- January has the highest monthly sales at $1,550
- Only 3 out of 20 products are high performers
- 9 out of 20 products are above average sales of $408

## 📁 Files
- `Sales_Analysis_Project.ipynb` — Main analysis notebook
- `sales_data.csv` — Dataset used

## 🔍 SQL Queries Covered
- GROUP BY + Aggregations
- CASE WHEN performance rating
- Subqueries
- LIMIT + ORDER BY
